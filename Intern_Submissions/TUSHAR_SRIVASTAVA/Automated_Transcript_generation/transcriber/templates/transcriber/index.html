About The Project

This platform is designed to automate podcast transcription using OpenAI's Whisper model.

  • Audio Ingestion: Upload whole podcast files.
  • Transcription: Accurate ASR using 'turbo' Whisper model.
  • Human-in-the-Loop: Review and edit transcripts before finalizing.
  • Export: Download finalized transcripts as PDF.
